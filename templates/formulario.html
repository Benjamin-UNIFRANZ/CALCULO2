Derivadas Parciales

Concepto Descripción Ejemplo
Definición La derivada parcial de una función multivariable respecto a una variable es la derivada de la función manteniendo las otras variables constantes.
Notación \(\frac{\partial f}{\partial x}\), \(f_x\), \(D_x f\)
Ejemplo Si \(f(x, y) = 3x^2 + 5y^3\) \(\frac{\partial f}{\partial x} = 6x\), \(\frac{\partial f}{\partial y} = 15y^2\)
Funciones trigonométricas \(\frac{\partial}{\partial x} \sin(x) = \cos(x)\), \(\frac{\partial}{\partial x} \cos(x) = -\sin(x)\)
Funciones exponenciales \(\frac{\partial}{\partial x} e^x = e^x\)
Casos especiales Derivadas parciales de funciones implícitas y derivadas de orden superior.

Integrales Triples

Concepto Descripción Ejemplo
Definición La integral triple de una función de tres variables se calcula integrando la función respecto a cada variable en un orden específico.
Notación \(\iiint_V f(x, y, z) \, dV\)
Ejemplo \(\iiint_V x y \, dV\) con \(V\) definido por \(1 \leq x \leq 2\), \(0 \leq y \leq x\), \(0 \leq z \leq y + 2z\)
Pasos
  1. Integrar respecto a \(z\)
  2. Integrar respecto a \(y\)
  3. Integrar respecto a \(x\)
Casos especiales Integrales en coordenadas cilíndricas y esféricas.

Integrales Dobles

Concepto Descripción Ejemplo
Definición La integral doble de una función de dos variables se calcula integrando la función respecto a cada variable en un orden específico.
Notación \(\iint_R f(x, y) \, dA\)
Ejemplo \(\iint_R (6x^2)y - 2x \, dA\) con \(R\) definido por \(1 \leq x \leq 4\), \(0 \leq y \leq 2\)
Pasos
  1. Integrar respecto a \(y\)
  2. Integrar respecto a \(x\)
Casos especiales Integrales en coordenadas polares.
